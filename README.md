# Dragon Radar for Hiccup's Legacy

## RU

Dragon Radar for Hiccup's Legacy — это utility-addon для Minecraft Forge 1.20.1 и Hiccups Legacy.

### Возможности
- поиск ближайших драконов
- поиск драконьих яиц
- локализация:
  - русский
  - английский
- книжка-гайд
- стартовый набор при первом входе в новый мир:
  - Dragon Radar
  - Dragon Field Guide
- стартовый набор работает и в survival, и в creative

### Для чего подходит
- одиночная игра
- приватный кооператив
- личные сборки

### Требования
- Minecraft 1.20.1
- Forge
- Hiccups Legacy 2.0.0

### Установка
1. Установить Forge 1.20.1
2. Установить Hiccups Legacy 2.0.0
3. Положить мод в папку `mods`

## EN

Dragon Radar for Hiccup's Legacy is a utility addon for Minecraft Forge 1.20.1 and Hiccups Legacy.

### Features
- nearby dragon detection
- dragon egg detection
- localization:
  - Russian
  - English
- guide book
- starter kit on first world join:
  - Dragon Radar
  - Dragon Field Guide
- starter kit works in both survival and creative

### Intended use
- singleplayer
- private co-op
- personal modpacks

### Requirements
- Minecraft 1.20.1
- Forge
- Hiccups Legacy 2.0.0

### Installation
1. Install Forge 1.20.1
2. Install Hiccups Legacy 2.0.0
3. Put the mod into the `mods` folder
